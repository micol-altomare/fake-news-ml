# Fake News Detection ML Application
A Flask app for fake news classification using a Naive Bayes approach deployed using AWS Elastic Beanstalk.

```
fake-news-ml/

│   ├── basic_classifier.pkl

├── Procfile
├── requirements.txt
└── readme.md
```
